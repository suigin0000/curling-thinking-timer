# curling-thinking-timer
A thinking timer that can be used in curling under the 10-end thinking time rule.

カーリングのシンキングタイマーです。


視認性を上げるために、大きなタイマー表示をしています。


タブレットで使うことを想定していますが、現在書きかけです。


カーリングのシンキングタイム制というルールで試合を行う時、タイマーを顧問の先生や特定の研究室から借りる必要がありました。そのため、誰でも気軽にシンキングタイマーを使ってシンキングタイム制のルールで試合ができるようにWebアプリ化してしまえということで作成を行ないました。


メインで動くタイマーをチーム分(2つ)、休憩時に動くタイマーを2種類、延長戦時のタイマーをチーム分(2つ)合計6つのタイマーが含まれます。


>・メインタイマー(10エンド → 38分、8エンド → 30分)
>  ・赤
>  ・黄色
>・エンド間(休憩時間 1分→ 投球時間 10秒)
>・ハーフタイム(休憩時間　5分→ 投球時間 10秒)
>・タイムアウト(2分30秒)
>・エクストラエンドタイマー(4分30秒)


実際にカーリングの公式戦やツアー大会の運営でアルバイトでタイマー計時のお手伝いやアルバイトがあるので、そこで使われているタイマーと似たようなデザインとなっています。そのためルールを知らない人でも誰にでも使いやすいというわけではありませんが、使い方やシンキングタイム制のカーリングのルールを先輩等から教われば大体の使い方がわかるようなUIを目指しました。


また、タイマーのスタートボタンを連打する等の予期しない動作をしても問題なく試合が続行できるような設計をしています。
